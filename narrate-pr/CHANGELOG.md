# Changelog

## 1.0.0 - 2026-07-13

- Added: `walkthrough.json` v1 schema (`references/schema.md`) and a zero-dependency validator (10 rules, all-violations reporting); deterministic renderer (`render.mjs`) with lane, sequence, and depmap diagram renderers, a generated per-package palette, dual themes, sha-pinned code receipts and PR-diff-anchored doc receipts, and a `--standalone` flag; golden fixture; full `SKILL.md` workflow (scout → fan-out research → edge verification → synthesize → validate/render → artifact publish) with a trust-boundaries table.
- Changed: depmap redesigned after live acceptance feedback — exemplar label grammar (no tag backgrounds, wrapped/beside-the-line labels), label-width-aware gutter/row sizing, border-anchored perpendicular edge stubs, gutter-centerline routing (no through-node or collinear legs), chips wrap inside node rects.
- Note: acceptance-tested end-to-end on a real 112-file PR; version gate passed by Chris 2026-07-13.
- Hardened (PR #17 review): validator rules 11-16 (doc/url/report receipt ref shapes, depmap layout bounds, required prefixed ids, review-order step contiguity), repo-relative code-ref enforcement, structure violations for non-object collection entries; renderer scheme-allowlists all hrefs (`javascript:`/`data:` rejected) and renders report receipts as local-only badges in fragment mode with an optional `--report-map` flag mapping reports to published artifact URLs; `node --test` suite (64 tests); SKILL.md gains dirty-tree checkout guard, PR identity fetch (`pr.repo`/`pr.branch`/`pr.base`), fenced-and-scanned file list, explicit `<output-dir>` paths, and `disable-model-invocation: true` so the skill only triggers via `/narrate-pr`.
- Hardened (PR #17 review, round 2): depmap `layout.nodes` now requires a positioned entry for every node (rule 6 bijection, validator surjectivity check) so no node goes undrawn; code/doc receipt refs reject Windows drive-letter and backslash paths; renderer percent-encodes GitHub blob-URL path segments (spaces/`#`/non-ASCII) while keeping the diff-anchor digest on the raw path; depmap-only architecture no longer emits an empty Architecture section or dead TOC link; `{component inventory}` and `{scope}` fan-out payloads are `<external_data>`-fenced; SKILL.md Step 2 budgets the PR payload (title+body ≤ 20 KB, file list ≤ 2000 entries). `node --test` suite now 74 tests.

## 0.1.0 - 2026-07-12

- Initial scaffold — skill directory, skeleton `SKILL.md` (frontmatter + outline, no step logic yet), and brief templates for the fan-out research and edge-verification subagent steps. No schema, validator, or renderer yet; those land in a later phase.
